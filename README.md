Modern template for anki flash cards


# Usage

1. Add additional fields
    - Transcription
    - Definition
    - POS (PartOfSentence)
    - Example1
    - ExampleTranslation1
    - Example2
    - ExampleTranslation2
    - Example3
    - ExampleTranslation3
2. Copy Style.css into you card style table
3. Copy Front.html and Back.html into Front and Back template settings
